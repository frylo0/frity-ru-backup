Крутим страницу.
Если секция прокручена - увеличиваем её на весь экран.
Если не прокручена - уменьшаем.
Надо наблюдать состояние скролла страницы, чтобы не бегать циклом по всем секциям.

Возможные состояния секции:
1. На весь экран (прокручена);
2. Уменьшенная (не прокручена).

Возможные переходы состояний:
1. Стала прокручена; == удалить класс поп, если он есть
2. Стала не прокручена; == добавить класс поп, если его нет
3. Ничего не поменялось. == ретурн

Сделать через события.
При скролле работаем с одной секцией.
Храним прошлое значение скролла.
Если при скролле меняется секция - запускаем обработчик для прошлой и текущей.
Обработчики: onScrollIn, onScrollOut.
